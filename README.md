Adaptive Study Planner

A brain-inspired adaptive study planner that personalizes study schedules based on learning patterns, quiz performance, mistakes, and revision habits.

Features

- Personalized study recommendations
- Adaptive study hour allocation
- Revision scheduling
- Performance tracking
- Learning pattern analysis
- Brain-inspired decision making

Technology Stack

- Python (Flask)
- HTML
- CSS
- JavaScript
- SQLite

Project Goal

The goal of this project is to create an intelligent study planning system that adapts to a student's learning behavior and provides personalized recommendations to improve learning outcomes.
